# Ren'Py AI Skills Collection

本仓库收集了一系列用于辅助 Ren'Py 游戏开发的 AI Skills，旨在通过 AI 提高开发效率。

## 使用说明与建议

在使用本仓库的 Skill 之前，请仔细阅读以下说明：

1.  **运行环境要求**

    - 这些 Skill 设计用于支持 **Agent 多步执行任务** 的 IDE 环境。
    - **推荐环境**：[Trae](https://www.trae.ai/) (可开箱即用，自动加载 Skill)。
    - **其他环境**：如果你使用的是其他 AI 辅助编程工具（如 Cursor、VS Code Copilot 等），可能需要手动将 Skill 的定义文件（通常在 `.trae/skills/` 目录下）加载为自定义 Skill，或者在每次对话时将其内容复制作为提示词（Prompt）发送给 AI。

2.  **模型选择至关重要**

    - Skill 的执行效果与底层使用的 **大语言模型（LLM）** 能力关系极大。
    - 不同的模型在代码生成、创意写作、翻译准确度上各有优劣。建议先进行小规模尝试，对比不同模型的效果，找到最适合你项目风格和个人喜好的模型。

3.  **成本与稳定性**

    - 大模型输出具有一定的随机性和不稳定性，可能需要多次尝试才能得到满意的结果。
    - 请注意 API 调用产生的持续开销，根据自身预算合理使用。

4.  **灵活调整，非魔法黑箱**
    - Skill 本质上是封装好的提示词和工作流，并非不可修改的“魔法黑箱”。
    - 强烈建议根据你自己的项目需求、代码规范和实际情况，灵活调整 Skill 的提示词内容（位于 `.trae/skills/` 对应目录下的 `.md` 文件中），以获得最佳效果。

### renpy-text2script 文本转剧幕 Skill

使用方法：

- **Trae 环境**：直接在对话中要求使用 `renpy-text2script` skill。
- **非 Trae 环境**：请引用或复制 [.trae/skills/renpy-text2script/SKILL.md](.trae/skills/renpy-text2script/SKILL.md) 文件内容作为提示词。

**核心功能：**

1.  **文本拆分与 Ren'Py 脚本编写**

    - 将原始文本拆分为多个独立的剧幕（Scene），每个剧幕对应一个 `.rpy` 文件。
    - 输出路径：`game/scripts/[章节名]/`（如 `game/scripts/ch4/`）。
    - 文件命名规范：`[章节]_[序号]_[描述].rpy`（如 `ch4_s1_meeting.rpy`）。
    - 代码规范：
      - 包含标准 Label 定义。
      - 自动声明未定义角色。
      - 转换对白为 Ren'Py 语法。
      - 添加详细的演出注释（场景、音乐、音效、表情、特效），并使用 Python 注释 `#` 标记。

2.  **资产需求设计**
    - 为每个剧幕生成配套的资产设计文档。
    - 输出路径：`game/assets/[剧幕名]_assets.md`。
    - 内容包含：背景、CG、BGM、SFX 的详细描述、搜索关键词及 AI 生成提示词。

**注意事项：**

1.  生成结果为仅供参考的草稿，不保证结果可直接运行，需要根据实际情况修改代码路径及补充静态资源。
2.  此工具主要用于快速搭建演出框架和梳理资源需求，约占 Ren'Py 脚本演出总工作量的 20%，剩余 80% 的精细化演出仍需人工调整。
3.  可根据自己经验和实际情况调整提示词，以取得更好的效果

### renpy-translate 自动翻译 Skill

使用方法：

1.  **第一步：初步翻译**：`@ch1` 翻译这些文件

    - 在对话中要求使用 `renpy-translate` skill 对指定的 `.rpy` 文件进行翻译。
    - Skill 会自动识别未翻译的条目并进行填充，同时建立术语表以保持一致性。

2.  **第二步：审查与修正（关键）**：`@ch1` 审查这些翻译内容是否有中式表达并修正
    - **强烈建议**在初步翻译完成后，再次要求 Skill 对已有翻译进行“审查”或“检查中式英语”。
    - 此步骤能显著修正直译、Chinglish 及语气不当的问题，大幅提高文本质量。

**免责声明：**

- 此工具生成的翻译仅作为**草稿**参考。
- AI 翻译无法完全理解复杂的语境、潜台词及文化细微差别。
- **不可替代专业翻译人员**。若用于商业发布，请务必进行人工校对，否则因翻译错误导致的后果自负。
