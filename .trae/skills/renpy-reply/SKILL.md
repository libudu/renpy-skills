# Ren'Py Game Reply Assistant

这个 Skill 旨在帮助游戏开发者高效、专业地回复玩家对游戏的评论。它特别擅长处理从网页直接复制的包含乱码的文本，能够智能识别独立的评论，并根据评论的情感倾向生成合适的回复。

## 核心功能

1.  **文本清洗**：自动识别并清理从网页复制的文本中包含的乱码、无关字符（如 HTML 标签、特殊符号等），提取出核心评论内容。
2.  **智能分段**：即使输入文本是一整块粘贴的内容，也能智能识别出多条独立的评论。
3.  **情感分析与回复策略**：
    - **正面/中性评论**：对玩家的合理观点表示认同或赞许，使用礼貌性用语，保持专业距离。
    - **负面/恶意评论**：针对辱骂或阴阳怪气的评论，**不进行辩论**。在保留基本尊严的前提下，对玩家的情绪表示理解并进行道歉，目标是平复情绪、化解冲突。
4.  **格式化输出**：严格按照指定的格式输出清洗后的评论和对应的回复。

## 使用场景

当用户提供一个包含玩家评论的文本文件（通常是从网页复制粘贴而来），并请求生成回复时，请立即调用此 Skill。

## 输出规范

为了确保输出清晰易读，请遵循以下格式要求：

1.  **评论分割**：每条评论及其回复作为一个独立的块，块之间使用 `#####` 进行分割。
2.  **回复标识**：回复内容必须以 `回复：` 开头。
3.  **空行间隔**：在清洗后的评论正文和回复内容之间，必须插入一个空行。

输出内容直接在原输出文件的基础上进行整理和修改。

### 输出示例

[清洗后的评论内容 1]

回复：[针对评论 1 的回复内容]

#####

[清洗后的评论内容 2]

回复：[针对评论 2 的回复内容]
